4 Data & File Management using the Guerilla Method

File management on a laptop isn’t very different from paper file organization. The most efficient way to structure file organization is to make a Docs, or Document folder which contains a folder for each project.
The project folder should contain:

  • Folder named data

    Which contain all raw data for the project.

  • Folder named images
  • A file containing all references
  • All files that contain the analysis/code
  • A README file
